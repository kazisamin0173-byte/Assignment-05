1️⃣ What is the difference between var, let, and const?
Ans: Difference between var, let, and const-
var: old way, can change value and declare again.
let: can change value, but cannot declare again in same block.
const: value cannot be changed and cannot declare again.

2️⃣ What is the spread operator (...)?
Ans: Spread operator (...) is used to expand or copy elements from an array or object.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:Difference between map(), filter(), and forEach()-
map(): returns a new array after modifying each element.
filter(): returns a new array with elements that pass a condition.
forEach(): loops through each element but does not return a new array.

4️⃣ What is an arrow function?
Ans: An arrow function is a shorter way to write a function in JavaScript using =>.

5️⃣ What are template literals?
Template literals are used to create strings easily in JavaScript using backticks ( ) and can insert variables using ${}.
